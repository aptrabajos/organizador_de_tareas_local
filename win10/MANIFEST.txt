🚀 OPCIONES DE INSTALACIÓN:

1. INSTALADOR MSI (Más fácil):
   - Doble click en el archivo .msi
   - Sigue el asistente
   - Listo!

2. SCRIPT POWERSHELL (Automatizado):
   - Abre PowerShell como Administrador
   - cd ruta\a\win10
   - .\scripts\install.ps1
   - Sigue las instrucciones

3. COMPILAR DESDE FUENTE (Para desarrolladores):
   - Instala: Node.js, Rust, pnpm
   - cd source
   - pnpm install
   - pnpm run tauri:build
   - Instala el MSI generado
